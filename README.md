# 📖 Titi - Lector Accesible PRO

**Titi** es una herramienta web frontend diseñada para democratizar el acceso a la información. Actúa como un lector inmersivo que ayuda a personas con dislexia, debilidad visual, TDAH o fatiga visual a consumir textos y documentos de manera cómoda, autónoma y personalizada.

## ✨ Características Principales

* **🗣️ Síntesis de Voz Sincronizada:** Utiliza la Web Speech API nativa para leer el texto en voz alta, resaltando visualmente cada palabra en tiempo real para facilitar el seguimiento visual.
* **🧠 Modo Dislexia:** Transforma la interfaz utilizando tipografías sin serifas (optimizadas para la dislexia), aumentando el interlineado y el espacio entre palabras/letras sobre un fondo de bajo estrés visual.
* **🌗 Modo Alto Contraste:** Interfaz oscura con colores de alto impacto para usuarios con baja visión o sensibilidad a la luz.
* **📄 Soporte de Documentos:** Permite cargar archivos `.txt` y extraer texto directamente de documentos `.pdf` de forma local en el navegador, gracias a la integración con `PDF.js`.
* **🎛️ Control Total:** Personalización de la voz del asistente, velocidad de lectura y tono, adaptándose al ritmo cognitivo de cada usuario.
* **📱 Diseño Responsivo:** Interfaz moderna (UI/UX) que se adapta perfectamente tanto a computadoras de escritorio como a dispositivos móviles.

## 🛠️ Tecnologías Utilizadas

Este es un artefacto digital ligero, rápido y seguro. Funciona al 100% en el lado del cliente (Frontend), lo que significa que los documentos del usuario nunca se suben a ningún servidor externo.

* **HTML5 & CSS3** (Diseño Neumórfico y Responsive)
* **JavaScript (ES6)**
* **Web Speech API** (Motor de lectura TTS)
* **PDF.js** (Procesamiento de documentos)

## 🚀 Cómo usarlo

Al ser una aplicación web estática, no requiere instalación.
1. Ingresa a la herramienta a través del enlace web.
2. Pega tu texto en el área principal o sube un documento (`.txt` o `.pdf`) desde el menú lateral.
3. Selecciona tus preferencias visuales (Dislexia / Contraste) y la velocidad de la voz.
4. Presiona **"Iniciar Lectura"** y disfruta de la experiencia inmersiva.
